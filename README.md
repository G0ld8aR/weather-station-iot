# Weather Station IoT

A custom ESP32-based weather station project focused on collecting local environmental data and presenting it on a low-power display, with room for future wireless expansion and field deployment.

## Project Overview

This project is a hands-on IoT weather station build centered around an ESP32 platform and modular sensor integration. The goal is to create a reliable, solar-friendly, field-ready system that can measure and display core weather data such as temperature, humidity, wind, and rainfall.

The build is being developed in stages, starting with bench testing and hardware validation, then moving into permanent wiring, enclosure design, display integration, and final outdoor deployment.

## Current Status

So far, this project has accomplished a few things:

- Initial repository setup and organization
- Arduino code workspace creation
- Diagram workspace creation
- Parts planning for the final permanent build
- E-ink display Arduino environment notes and working settings
- Hardware troubleshooting and validation for core weather station components
- Pin mapping, wiring refinement, and breadboard-to-permanent-build planning
- Enclosure and field-mount planning for sensors and controller hardware

## Main Goals

- Measure environmental conditions using external sensors
- Display weather data locally on an e-ink screen
- Build a clean, serviceable, semi-permanent controller assembly
- Support outdoor deployment with weather-conscious enclosure design
- Prepare the system for future wireless features such as LoRa or BLE if needed
- Document the full build so it can be reproduced and improved later

## Planned / In-Progress Features

- Temperature and humidity sensing
- Wind speed input
- Wind direction input
- Rainfall sensing
- E-ink display output
- Solar/battery-ready power planning
- Modular RJ11-based weather sensor integration
- Expandable architecture for future communication features
